# Images Directory

Place your website images here:

- **logo.png** or **logo.svg** - Company logo for the navbar
- **hero-background.jpg** - Optional hero section background image
- **favicon.ico** - Website favicon

## Recommended Image Sizes:

- Logo: 200x60px (transparent background)
- Hero background: 1920x1080px
- Service icons: Can use SVG inline icons (already included in HTML)
- Favicon: 32x32px or 64x64px

## Image Optimization Tips:

1. Use WebP format for better compression
2. Optimize images using tools like TinyPNG or ImageOptim
3. Use SVG for logos and icons when possible
4. Compress JPEG/PNG images to reduce file size
